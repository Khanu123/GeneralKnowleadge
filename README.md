# General Knowledge Quiz

A Python command-line quiz game that asks multiple-choice questions, validates answers, tracks guesses, and calculates a final score.

## Overview

This project demonstrates how to structure a small interactive program using collections, loops, conditional statements, and user input. It is a clean beginner project that can be extended into a larger quiz engine.

## Skills Demonstrated

- Python lists and tuples
- Input normalization
- Control flow
- Score calculation
- Terminal-based user interaction

## How to Run

```bash
python main.py
```

If the script has a different filename, run that Python file directly.

## What I Would Improve Next

- Store questions in a JSON or CSV file.
- Add categories and difficulty levels.
- Add unit tests for scoring.
- Track high scores across sessions.
